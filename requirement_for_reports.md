## Отчет 1: Финансово-аналитический отчет "Эффективность работы мастеров и услуг"

**Назначение отчета:** Отчет предназначен для анализа финансовых показателей салона, оценки производительности мастеров и популярности услуг за выбранный период.

**Бизнес-задачи:**
1. Определить самых доходных мастеров
2. Выявить наиболее популярные услуги и категории
3. Проанализировать средний чек по разным направлениям
4. Оценить активность мастеров (количество обслуженных клиентов)

**Используемые таблицы и поля:**
- clients: client_id, фио клиента
- masters: master_id, фио мастера, is_active
- services: service_id, название услуги, категория, базовая цена
- bookings: booking_id, price, datetime, status

**Ключевые метрики:**
1. Выручка мастера (сумма по price)
2. Количество обслуженных клиентов (уникальных client_id)
3. Количество выполненных услуг (count bookings)
4. Средняя цена услуги по мастеру/категории
5. Период активности (первая и последняя запись)

**Входные параметры:**
- Период (начальная и конечная дата)
- Статус записи (по умолчанию 'выполнена')
- Активность мастера (по умолчанию true)

**Выходные данные (колонки):**
1. ID и ФИО мастера
2. Категория услуги
3. Название услуги
4. Количество уникальных клиентов
5. Общее количество записей
6. Общая выручка
7. Средний чек
8. Даты первой и последней записи

**Сортировка:**
- По убыванию выручки
- По убыванию количества записей

## Отчет 2: Клиентская аналитика "Портрет и предпочтения клиентов"

**Назначение отчета:** Анализ клиентской базы для выявления лояльных клиентов, их предпочтений и паттернов поведения.

**Бизнес-задачи:**
1. Сегментировать клиентов по уровню лояльности
2. Определить предпочтения каждого клиента
3. Выявить VIP-клиентов
4. Проанализировать частоту посещений
5. Понять клиентские пути (какие услуги и у каких мастеров выбирают)

**Используемые таблицы и поля:**
- clients: Все поля (включая дату регистрации)
- masters: master_id, фио мастера
- services: service_id, категория
- bookings: Все поля (включая price, datetime, status)

**Ключевые метрики:**
1. Общая сумма потраченных средств
2. Количество посещений
3. Количество уникальных дней посещений
4. Любимый мастер (по количеству посещений)
5. Любимая категория услуг
6. Разнообразие услуг (сколько разных услуг попробовал)
7. Период клиентства (от первой до последней записи)

**Выходные данные (колонки):**
1. ID клиента и полное ФИО
2. Контактный телефон
3. Дата регистрации
4. Общее количество посещений
5. Общая сумма потраченного
6. Средний чек
7. Любимый мастер
8. Любимая категория услуг
9. Количество использованных разных услуг
10. Количество посещенных разных мастеров
11. Даты первого и последнего визита

**Сортировка:**
- По убыванию общей суммы потраченного
- По убыванию количества посещений

## Отчет 3: Операционный отчет "Загрузка и эффективность работы салона"

**Назначение отчета:** Мониторинг операционной эффективности, анализ загрузки мастеров и распределения услуг во времени.

**Бизнес-задачи:**
1. Оптимизировать расписание мастеров
2. Выявить пиковые и простаивающие часы
3. Оценить загрузку каждого мастера
4. Проанализировать длительность услуг
5. Понять распределение услуг по дням недели и времени суток

**Используемые таблицы и поля:**
- clients: client_id (для подсчета уникальных клиентов)
- masters: Все поля (включая is_active)
- services: Все поля (особенно duration, category)
- bookings: Все поля (особенно datetime, price, notes)

**Ключевые метрики:**
1. Загрузка мастера в часах (сумма duration)
2. Коэффициент заполненности (% от рабочего дня)
3. Количество записей в разрезе времени
4. Выручка за период
5. Процент повторных клиентов у мастера
6. Распределение по времени суток
7. Анализ заметок/особых пожеланий

**Временные срезы:**
- Разбивка по неделям
- По дням недели
- По часам суток

**Выходные данные (колонки):**
1. Начало недели (дата)
2. День недели
3. Час дня
4. ID мастера и краткое ФИО
5. Статус активности
6. ID и название услуги
7. Категория услуги
8. Длительность услуги
9. Количество записей
10. Общая загрузка в часах
11. Выручка за период
12. Средняя цена услуги
13. % загрузки мастера
14. Количество уникальных клиентов
15. % удержания клиентов
16. Частые заметки/пожелания

**Сортировка:**
- По дате (новые недели сверху)
- По дню недели и часу
- По загрузке мастера
